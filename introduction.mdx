## Welcome to TeraBox Auto

TeraBox Auto is a Telegram bot that automates the distribution of TeraBox links across multiple channels. It processes photo posts with TeraBox URLs, formats them with custom templates, and distributes them to configured channels while managing engagement through random post features.

## Key features

### Multi-channel distribution

Automatically forward TeraBox content to all configured channels with a single post. The bot manages channel lists through Google Sheets integration.

### Smart link detection

Validates and extracts TeraBox URLs from multiple domains:
- terabox.com
- nephobox.com
- freeterabox.com
- teraboxapp.com
- And 6 more supported domains

### Google Sheets database

Leverages Google Sheets as a lightweight database for:
- Channel management
- Post tracking
- User authorization
- Configuration storage

### Flask webhook integration

Runs as a Flask web server with Telegram webhook support, making it perfect for deployment on platforms like Render or Heroku.

## How it works

<Steps>
  <Step title="Authorized user sends photo with caption">
    The bot listens for photo messages from authorized users containing TeraBox links in the caption.
  </Step>
  <Step title="Link validation and extraction">
    The bot extracts and validates the TeraBox URL against the list of supported domains.
  </Step>
  <Step title="Post formatting">
    Content is formatted using customizable HTML templates with inline keyboard buttons.
  </Step>
  <Step title="Multi-channel distribution">
    The formatted post is sent to the main channel and all configured channels from the Google Sheets database.
  </Step>
</Steps>

<Note>
  TeraBox Auto requires Python 3.11 and active Google Sheets API credentials to function properly.
</Note>

## Use cases

**Content distribution networks**: Manage multiple Telegram channels from a single source.

**Media sharing**: Distribute TeraBox-hosted content with consistent formatting across your channel network.

**Engagement automation**: Use random post features to keep channels active with historical content.
