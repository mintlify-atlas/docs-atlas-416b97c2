This guide will help you deploy TeraBox Auto and start distributing content across your Telegram channels.

## Prerequisites

Before you begin, ensure you have:

- Python 3.11 installed
- A Telegram bot token from [@BotFather](https://t.me/BotFather)
- Google Sheets API credentials (JSON file)
- A Google Sheet named "TeraX2nd" with two worksheets: "general" and "channels"

## Setup steps

<Steps>
  <Step title="Clone and install dependencies">
    Install the required Python packages:

    ```bash
    pip install Flask Gunicorn pyTelegramBotAPI random2 gspread oauth2client
    ```

    <Info>
      The bot uses `pyTelegramBotAPI` (also known as telebot) for Telegram integration and `gspread` for Google Sheets access.
    </Info>
  </Step>

  <Step title="Configure Google Sheets credentials">
    Place your Google Sheets API credentials file in the project root:

    ```bash
    # Rename your credentials file to match the code
    mv your-credentials.json Badnaamchora2784.json
    ```

    Create a Google Sheet named **TeraX2nd** with two worksheets:

    - **general**: Stores configuration (cell B1 for last post ID, cell B2 for post channel ID)
    - **channels**: Stores channel IDs (column A for row numbers, column B for channel IDs, cell A51 for counter)

    <Warning>
      Ensure the service account email from your credentials has edit access to the Google Sheet.
    </Warning>
  </Step>

  <Step title="Update configuration values">
    Edit `config.py` with your settings:

    <CodeGroup>
      ```python config.py
      POSTCHANNEL = -1002191085367  # Your main channel ID
      GENERALCHANNEL = "https://t.me/+yourChannelLink"  # Your channel invite link
      PostText = """<b>{}🥰
        
      {}
      {}
        
      🛑Login to watch full video🛑
      ━━━━━━━━━━━━━━━━━━━━━
      ✅ ᴊᴏɪɴ ɴᴏᴡ ꜰᴏʀ ᴍᴏʀᴇ ᴠɪᴅᴇᴏꜱ!
      {}
        </b>"""
      AuthUser = [5507466970, 1958848922]  # Add your Telegram user IDs
      AdText = """<a href='{}'>🔞 Join Our Private Channel
      🔞 Join Our Private Channel</a>"""
      ```
    </CodeGroup>

    Replace the values:
    - `POSTCHANNEL`: Your main channel ID (get it using bots like @userinfobot)
    - `GENERALCHANNEL`: Your channel invite link
    - `AuthUser`: List of Telegram user IDs authorized to use the bot
  </Step>

  <Step title="Update bot token and webhook URL">
    Edit `app.py` and replace the API token and webhook URL:

    ```python app.py
    API_TOKEN = 'YOUR_BOT_TOKEN_HERE'  # From @BotFather
    
    # In the webhook() function:
    bot.set_webhook(url='https://your-app-url.onrender.com/' + f"{API_TOKEN}")
    ```

    <Note>
      The webhook URL should match where you deploy the app (Render, Heroku, etc.)
    </Note>
  </Step>

  <Step title="Run the bot">
    Start the Flask server:

    ```bash
    python app.py
    ```

    The bot will run on port 1000 by default. For production deployment:

    ```bash
    gunicorn app:server -b 0.0.0.0:$PORT
    ```
  </Step>

  <Step title="Set up the webhook">
    Visit your deployment URL in a browser to initialize the webhook:

    ```
    https://your-app-url.onrender.com/
    ```

    You should see "!uk" confirming the webhook is set.
  </Step>
</Steps>

## Test your bot

<Steps>
  <Step title="Verify bot is running">
    Send `/start` to your bot on Telegram. If you're an authorized user, you should receive "Runnung.." (the typo is in the source code).
  </Step>

  <Step title="Add channels to distribution list">
    Use the `/add` command followed by channel IDs:

    ```
    /add
    -1001234567890
    -1009876543210
    ```

    Each valid channel ID will be added to your Google Sheet.
  </Step>

  <Step title="Post content">
    As an authorized user, send a photo with a caption containing a TeraBox link:

    ```
    Amazing Video Collection
    
    https://terabox.com/s/1abc123xyz
    
    Check it out!
    ```

    The bot will:
    1. Extract the TeraBox URL
    2. Format the post with your template
    3. Send it to the main channel and all configured channels
    4. Reply with "Done❤️"
  </Step>
</Steps>

## Common issues

<Warning>
  **"you are not authrise" message**: Add your Telegram user ID to the `AuthUser` list in `config.py`.
</Warning>

<Warning>
  **Google Sheets errors**: Verify that your service account has edit permissions on the TeraX2nd sheet and that all required worksheets exist.
</Warning>

<Warning>
  **Webhook not working**: Ensure your deployment URL is HTTPS and publicly accessible. Check that the webhook URL in `app.py` matches your deployment.
</Warning>
